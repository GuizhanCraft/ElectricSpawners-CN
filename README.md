# ElectricSpawners 电力刷怪笼

ElectricSpawners 是一个 Slimefun4 附属插件。
你需要安装 Slimefun4 才能让此插件运行。

ElectricSpawners 让你可以将已修复的刷怪笼制作成电力刷怪笼。
电力刷怪笼与普通刷怪笼功能类似，但其依赖电力工作，且不需要有玩家在附近。

[了解更多...](https://slimefun.guizhanss.wiki/Addons)
